# VHC Authorisation Sync Testing

1. From VHC Summary (red/amber), authorise a VHC item.
2. Confirm Customer Requests -> Authorised VHC Items shows it and Write Up -> Rectification shows it.
3. From Parts Authorised, remove or decline the linked part(s).
4. Confirm the item disappears from Customer Requests and Write Up.
5. Refresh the job card and confirm the state remains consistent.
