# VHC Summary Status Manual Checklist

1) Create a new job with VHC required.
2) Add one red VHC item via the technician VHC builder.
3) Open the manager VHC Summary tab.
4) Confirm the item appears in the Red section with status "Add labour & parts".
5) Add labour hours for the row and confirm status changes to "Add parts".
6) Add parts (or mark parts not required) and confirm status changes to "Awaiting customer decision".
7) Authorise the row and confirm it moves to the Authorised section with green status dot.
8) Reset the row back to pending and confirm it returns to the Red section.
